# Accessing Our Clusters Using Web Browsers

As of spring 2026: we have two clusters (JHPCE & JADE) and two kinds of web access -- JHPCE uses a custom solution and JADE uses an [Open OnDemand (OOD)](https://www.openondemand.org) server. During 2026 we hope to stand up an OOD server for the JHPCE cluster.

## Main JHPCE Cluster

We have a web portal named [jhpce-app02.jhsph.edu](https://jhpce-app02.jhsph.edu) configured to offer a number of services. Click on the links to learn more.

* Reset your password or generate a OTP ([learn more](../portal/web-reset.md)) (**{==requires a valid JHEDID account==}**)
* Run applications on the cluster (RStudio, JupyterLab, VS Code) ([learn more](../portal/web-apps.md))
* Inspect a catalog of research databases (under development) ([learn more](../portal/datacatalog.md))

For security reasons, the JHPCE web portal is only available to computers on Hopkins networks. If you are not on a Hopkins campus, that means that you need to use the VPN to be able to see that node.

## JADE Cluster

JADE is the JHPCE Advanced Data Enclave. One needs a different user account to access it.

Our Open OnDemand server[jade-ondemand01.jhsph.edu](https://jade-ondemand01.jhsph.edu) is under development in April 2026.
