Dear JHPCE and JADE Communities,

The DCS10 storage array will be offline on Wednesday, July 8, from 6:00 PM until 9:00 PM. We have been seeing occasional lockups on DCS10 and have determined that a kernel update is needed to resolve the issue. During this maintenance window, we will install the kernel update along with other software updates.

While DCS10 is offline, access will be unavailable to all JHPCE directories under /dcs10 as well as all JADE data directories under /data (including /data/cms/, /data/dbgap/, and /data/nist800171/).

Please plan to avoid running jobs against these directories during the maintenance window. Jobs that are using these paths may simply pause while the storage is unavailable, but there is also a possibility that they could terminate.

This maintenance will only affect data stored on the DCS10 storage array. All other storage will remain available, and other jobs should not be impacted.

Please let us know if you have any questions
