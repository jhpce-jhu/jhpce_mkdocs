    In our testing, the GPU version took about 50 minutes to run, and the non-GPU version took nearly 2 hours.

    You can use run_alphafold.sh --help to see all of the options for running alphafold.

    You do also need a fair amount of RAM. In the above exmaples, we requested 100GB, and in looking at the sacct output for these example jobs, only used about 17GB for the non-gpu run and 34GB for the gpu run.

    To submit an alphafold job, you would create a SLURM batch file:

    [login31 /users/mmill116/alphafold]$ cat alpha1.sh
#!/bin/sh
#SBATCH --mem=100G
#SBATCH --cpus-per-task=8
#SBATCH --gpus=1
#SBATCH --partition=gpu

date
module load conda
conda activate alphafold-gpu
mkdir /tmp/mm-alphfold1 
run_alphafold.sh -d /legacy/alphafold/data -o /tmp/mm-alphfold1 -f ./test.fasta 
-t 2020-05-14 -n 8
date
    You would then submit the job with "sbatch", perhaps adding additional GPU options 
    [login31 /users/mmill116/alphafold]$ sbatch --gres=gpu:tesa100:1 alpha1.sh
