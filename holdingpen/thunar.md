# THUNAR

## Overview
Thunar is an X11 file manager with many features. Here we provide some tips for
using it during an interactive job session.

## Launching it
It is helpful to put it in the background so you can continue to use the same
terminal window for other things. You would do so with "thunar &"

Like many X11 programs, thunar will emit a number of error messages every time
you start it.  These can be ignored. They occur because our compute nodes do not
have graphics cards in them.

```
No matching fbConfigs or visuals found
failed to load driver: swrast
```
